## 一、项目简介

### 1. 项目概述

开发一款面向甜品店运营管理人员的甜点门店后台管理系统，聚焦甜品品类管控、甜品上下架状态维护、系统操作日志记录与行为追溯、员工信息运维、原料采购管理全业务流程，解决甜品店人工台账混乱、甜品库存状态无法自动更新、系统操作无日志留存、故障难以排查追溯、员工信息无系统化管理、原料采购无统一记录管理的行业痛点，实现甜品门店日常运营数字化、规范化、智能化、可追溯管理。

### 2. 技术选型

**前端**
- 框架：Vue3 (3.5) + TypeScript
- UI 组件库：Element Plus (2.x)
- 构建工具：Vite (6.x)
- 路由：Vue Router (4.x)
- 样式：原生 CSS
- 网络请求：Axios

**后端**
- 核心：SpringBoot(3.x) + SpringMVC
- MyBatis-Plus (3.x)
- 分页工具：PageHelper
- 开发工具：Lombok
- 语言：Java (21)
- 框架：SpringBoot(3.0.5)
- 构建工具：Maven (3.9.x)
- 数据库：MySQL 8.0
- 日志：SLF4J + log4j2
- 加密算法：BCrypt算法
- 代码版本控制：Git + Gitee（托管）
- 测试工具：Postman
- 单元测试框架：JUnit5

**开发工具**
- IntelliJ IDEA 2025.x
- WebStorm 2025.x

## 二、项目目标

### (一) 功能目标

1. **基础核心功能**
   - 实现系统用户注册、登录身份认证模块
   - 完成甜品分类管理、甜品信息列表展示基础功能开发

2. **扩展业务功能**
   - 甜品管理模块：新增甜品下架、售罄状态管理功能；售罄状态支持次日定时自动更新为正常在售；新增已下架甜品独立管理页面，支持单品手动上架、批量一键上架操作
   - 新增员工管理页面：实现员工信息新增、编辑、删除、查询、状态启用/禁用全生命周期管理
   - 新增原料采购页面：实现甜品制作原料信息录入、采购记录添加、采购列表查询、原料库存登记与管理功能
   - 新增系统扩展功能：新增日志文件管理功能，记录用户操作、接口调用、系统运行状态，便于问题排查与行为追溯

3. **通用功能**
   - 实现系统数据分页查询、信息增删改查、状态标识、数据持久化存储与后台统一管理

### (二) 技术目标

熟练掌握前后端分离开发模式、SSM接口开发、Vue3前端页面搭建、MySQL数据表设计与关联、定时任务配置与执行、前端状态管理与页面权限控制、独立子页面路由跳转等核心技术。

### (三) 成果目标

完成功能完整、运行稳定、界面友好的甜点管理系统；产出完整源码、数据库脚本、接口文档、测试用例、需求分析、项目计划书、系统日志文件等全套交付材料，满足课程验收与答辩要求。
